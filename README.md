## This project was developed during the [Driven](https://www.driven.com.br/) bootcamp, where I learned, among other things, how to work with React:

- It was my first project using Node.js and React
- Application componentization
- Importing and exporting components
- Using useState to render dynamic variables
- Rendering repeated components using arrays, map and props

## 📑 The Application:

Access this application at the following link: [https://instagram-sage-seven.vercel.app/](https://instagram-sage-seven.vercel.app/)

In this application, you can:

- Change the user's profile image
- Change the username
- Save posts
- Like posts with the like button or double click
- Follow recommended profiles
